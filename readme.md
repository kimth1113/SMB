## 팀원

- 김태현
  - Back-End 담당 : Django
- 박준수
  - Front-End 담당 : Vue

## SMB 개요

> TMDB API를 활용하여 최신 영화 정보를 수집하고, 검색 및 순위 조회 기능과 게시판 CRUD 기능을 구현한 첫 프로젝트입니다. 기간은 약 일주일 정도 소요되었습니다.
